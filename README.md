# Point Cloud Detection

Проект посвящён детекции объектов в облаках точек для indoor-сцен. На вход подаются LiDAR-данные с iPad/iPhone, по ним восстанавливается point cloud и используется как вход для модели детекции


## Ход работы

- Реализовано получение LiDAR-данных с iPad/iPhone
- Реализован пайплайн преобразования LiDAR-данных в point cloud
